## 資料集
我們本次專案資料集包含預訓練資料(台達閱讀資料集)與Fine-tuning(判決書資料集)，
並將資料集依照SQuADv1.1的格式去統一化，方便後續訓練模型。

### SQuAD格式
格式如下
```
{
    "data":[
        {   
            #文章內文
            "context": "國立中興大學（簡稱興大、NCHU），是位於臺中的一所高等教育機構。",
            
            #問題
            "question": "中興大學在哪裡",
            
            #題號
            "id": "1-1",
            
            #答案
            "answers": {

                #答案起始位置(根據文章內文計算)
                "answer_start": [
                    22
                ],

                #答案字串
                "text": [
                    "臺中"
                ]
            }
        },
        {
            "context": "鴻海科技集團是由臺灣企業家郭台銘創辦的跨國企業，總部位於臺灣新北市土城區，主要生產地則在中國大陸，以富士康做為商標名稱。",
            "question": "鴻海集團總部位於哪裡？",
            "id": "1-2",
            "answers": {
                "answer_start": [
                    30
                ],
                "text": [
                    "新北市土城區"
                ]
            }
        }
    ]

}
```
