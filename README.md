## Overview

## 資料集
我們本次專案資料集包含預訓練資料(台達閱讀資料集)與Fine-tuning(判決書資料集)，
並將資料集依照SQuADv1.1的格式去統一化，方便後續訓練模型。

## 判決書資料
根據判決書的裁定要件，我們設計不同的要件問題。

比如我們這次使用的判決書皆是偽照文書，根據偽照文書的必要三個條件
1. 被告人偽造了什麼文書?
2. 被告人做了甚麼偽造或冒用的行為?
3. 被告人損害了什麼利益?

用這三個問答問題，判決書的內文以及人工標註的答案來當作我們的訓練與測試資料集
